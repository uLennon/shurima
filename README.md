## :page_with_curl:Sobre
- Shurima é um sistema web que permite consultar informações do perfil de contas no League of Legends.

## :bookmark_tabs:Dependências

Este projeto foi construido usando:

- Java 17
- Spring Boot 3.2.1
- Lombok
- Thymeleaf
